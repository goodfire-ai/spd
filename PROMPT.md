# PROMPT.md

This file provides further guidance to Claude Code (claude.ai/code) when working with code in this repository.

When making new scripts and plots, you should think carefully about the best place to put them in the repo structure. Use the appropriate directory, and if it makes sense to create a new one, do so.

At the start of a Claude Code session, before you make any other changes to the code, you should create a new branch that you will work in. Your branch should have the same name as the current branch with "-claude" appended to the end. DO NOT COMMIT TO ANY BRANCH OTHER THAN THAT ONE DURING THE SESSION. You should commit your changes periodically as you work, including after the user approves your work when providing you with input. This way, it will be easy to track changes and will obviate the need to make multiple new files when a modification to an existing file would do.
